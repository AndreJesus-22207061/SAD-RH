# SAD-RH
Projeto no âmbito da disciplina de Engenharia de Requesitos e Testes

Criadores: André Jesus / Tomás Nave / Miguel Lourenço / Vasco Pereira / Nuno Barros 

Descrição:

O objetivo deste projeto no âmbito da disciplina de Engenharia de Requisitos e Testes foi realizar o planeamento completo de um projeto de desenvolvimento de software, neste caso um Sistema de Avaliação de Desempenho de Recursos Humanos [SAD-RH] de base digital.
Começamos com o planeamento clássico onde tivemos de desenvolver um Project Charter e um Scope Statement. Onde desenvolvemos também os requisitos e as matrizes de requisitos do projeto.
Depois na segunda parte do projeto passamos ao desenvolvimento de um diagrama de Gant , dos Use Cases e de uma estimativa de esforço.
Na terceira parte passamos ao desenvolvimento do modelo Agile onde desenvolvemos o scrum e os sprints de scrum.
Por fim na quarta parte do projeto desenvolvemos os Test Cases e uma arvore de falhas com o objetivo de desenvolver a analise de falhas.
Com este projeto aprendemos todos os passos necessários para planeara a arquitetura de um software.
